### SpaceSomething
A small Space Invaders inspired game done as qualification task for [WWW12](http://warsztatywww.pl/qualProblems/rozwoj-gry-tower-defense/)

## Build instructions
```sh
$ git clone https://github.com/krzys-h/SpaceSomething.git
$ cd SpaceSomething
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./SpaceSomething
```
